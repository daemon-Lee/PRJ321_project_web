# Hướng dẫn cơ bản về git

tải project về:
  git clone (link)
  
thêm file đã chỉ sữa:
  git add (file or *)

xác nhận thay dổi 
  git commit -m "thông điệp xác nhận thay đổi"
  
đẩy code lên project:
  git push origin (vai trò)
  
cập nhật thay đổi từ project
  git pull
